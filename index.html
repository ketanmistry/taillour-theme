What is a boilerplate theme?

Taillour is a Shopify 2.0 boilerplate theme built for web designers who want a theme that's easy to build custom stores with.

This type of theme is sometimes called a "Skeleton theme".

Taillour contains the essential templates, sections and features, each with a handful of useful settings. It gives you the flexibility to expand on them, giving you exactly what you need for your custom store.

Why use a boilerplate theme?

Retrofitting new features to pre-made themes is incredibly messy and difficult as the codebase is overcrowded with template settings, nested snippets and components.

And, most themes aren't made for customising.

What you can do with Taillour?

Integrate your own design

Design your client's store exactly as you need and on-brand. You can create your own layouts using the tools you're already used to.

Add new features easily

Add the features you need without having to worry about conflicting code. Taillour is coded simple, so you can extend as required without having to forage through countless nested files.

Only use the features you need

Every project is different, so you can build your client's store based on what is needed, keeping things clean and simple for you and your client.

Features

Taillour is an installable boilerplate theme that comes with common section templates and features that you can customise to your projects requirements. And, it comes with some additional features.

View demo store

Purchase together Bonus

Allow customers to purchase items together similar to Amazon. Assign items per product to get those extra sales.

Offer gift wrap Bonus

Offer gift wrap directly in the cart for any applicable products. All you have to do is create a Gift wrap product and tag those products you'd like to offer a gift wrap service on.

Recently viewed Bonus

The recently viewed section remembers the products customers have looked at and is fully customisable.

"Spend more" cart message Bonus

The cart has options to display a "spend more" message to show text like "spend ¬£2.00 more to get free delivery".

Accurate breadcrumbs Bonus

Breadcrumbs through the theme follow the menu structure of the site so they are always accurate.

JSON structured data Bonus

Schemas for products, collections, breadcrumbs, business, blogs and articles are included.

No-nonsense variant selector

An easy to edit variant selector with inline variant images, built entirely with liquid and basic JavaScript. Unlike Dawn, this version has no complex or nested JavaScript components giving you complete control.

Newsletter signup popup

Taillour features a newsletter popup complete with settings. It connects directly to Shopify's marketing tools.

Accessibility Bonus

Some basic accessibility, like keyboard navigation focus, has been included which you can improve on.

Announcement bar

No store goes without the standard announcement bar.

Predictive search

Shopify's predictive search comes as standard, with easy to change code so you can adjust as required.

Product recommendations

Shopify's product recommendations is included as standard and easy to adjust to fit your requirements.

Unrestricted HTML/CSS and Javascript

The theme comes with a basic stylesheet which you can trash or modify - it's completely up to you and your project. Vanilla javascript is used, so no dependencies are required.

Subcollections Bonus

A common request by clients is to display subcollections of a collection, and this is built-in to the theme.

Product image slider

A simple CSS (with minimal JS) image slider on the the product details page that's easy to tweak. The slider also displays the thumbnails.

Built-in contact form

The built-in contact form section allows you to create as many forms as you need, all with customisable form fields.

Optimised for Core Web Vitals

Taillour is optimised for web performance and core web vitals. Stylesheets are split, loading only when required, and Javascript is kept to a minimum.

Search engine optimised

Taillour follows SEO best practices, with all templates being coded using semantic HTML. Everything is included in the boilerplate from meta tags to JSON structured data snippets.

Is Taillour pre-styled?

Taillour has very basic black and white styling and is not meant to be used like this out-of-the-box.

I'm a designer, can I use this?

Yes! If you are comfortable working with HTML, CSS and a little bit of Liquid code, Tailour is suitable for you.

My client has asked for a custom feature. Is this easy to do with Taillour?

Yes, this is what Taillour was made for! You'll find it much easier to develop your custom feature in to Taillour as the codebase is purposefully clean and simple.

Can I use a particular framework such as Tailwind CSS, Vue or React?

Absolutely! You can use whichever libraries and frameworks you want. There are no dependencies built into Taillour.

How quickly could I build my client's store using this?

I've built stores within 4 to 5 days from scratch using Taillour, excluding the design stage of course.

Who is Taillour theme suitable for?

Taillour is suitable for anybody who can code HTML/CSS, JavaScript and Liquid. You don't need to be an expert, but should have good knowledge of the basics at least.
